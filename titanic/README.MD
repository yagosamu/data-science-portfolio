### Titanic Survival Analysis

This project performs an end-to-end data analysis and machine learning pipeline on the Titanic dataset.

### Steps:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Feature Importance Analysis

### Results:
- Best model: Random Forest (~83% accuracy)
- Key features: Fare, Age, Title, Sex

### Tools:
- Python
- Pandas
- Scikit-learn
- Seaborn / Matplotlib
